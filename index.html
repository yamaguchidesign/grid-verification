CSS Grid 基本理解ガイド

1. 基本的なグリッド

display: grid でグリッドコンテナを作成し、grid-template-columns で列を定義します。 1fr は利用可能な空間を等分する単位です。
/* 親要素（グリッドコンテナ） */ display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 10px; /* 子要素（グリッドアイテム） */ background: #3498db; color: white; padding: 20px; text-align: center; border-radius: 5px; font-weight: bold;

2. 異なる列サイズ

固定サイズ（200px）、可変サイズ（1fr）、固定サイズ（100px）を組み合わせて柔軟なレイアウトを作成できます。
grid-template-columns: 200px 1fr 100px;

3. 行の指定

grid-template-rows で行の高さを指定できます。アイテムが3行に配置されています。
grid-template-rows: 100px 150px 80px;

4. グリッドエリア（レイアウト作成）

grid-template-areas で視覚的にレイアウトを定義し、各アイテムに grid-area でエリア名を指定します。
grid-template-areas: "header header header" "sidebar main main" "footer footer footer";

5. 自動配置（レスポンシブ）

repeat(auto-fit, minmax(200px, 1fr)) で画面サイズに応じて自動的に列数が調整されます。 最小200px、最大は利用可能な空間を等分します。
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

個別のアイテムに grid-column: span 2 や grid-column-start: 2 で配置を制御できます。
.span-2 { grid-column: span 2; } .start-2 { grid-column-start: 2; }

主要なプロパティ一覧

コンテナ（親要素）のプロパティ：
• display: grid - グリッドコンテナにする
• grid-template-columns - 列の定義
• grid-template-rows - 行の定義
• grid-template-areas - エリア名でレイアウト定義
• gap - グリッドアイテム間の間隔

アイテム（子要素）のプロパティ：
• grid-column - 列の範囲指定
• grid-row - 行の範囲指定
• grid-area - エリア名の指定
• justify-self - 水平方向の配置
• align-self - 垂直方向の配置
